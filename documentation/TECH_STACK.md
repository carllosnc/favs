# Technology Stack

**Favs** is built with a modern stack focusing on performance and developer experience.

## Core
- **[Next.js 15](https://nextjs.org/)**: The React Framework for the Web.
- **[React 19](https://react.dev/)**: The library for web and native user interfaces.
- **[Bun](https://bun.sh/)**: A fast all-in-one JavaScript runtime.

## Styling
- **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework.
- **[Shadcn UI](https://ui.shadcn.com/)**: Re-usable components built using Radix UI and Tailwind CSS.
- **[Lucide React](https://lucide.dev/)**: Beautiful & consistent icons.

## Database & Backend
- **[Turso](https://turso.tech/)**: Edge-hosted, distributed database based on libSQL.
- **[Drizzle ORM](https://orm.drizzle.team/)**: TypeScript ORM that runs exactly how you think.

## Authentication
- **[Better Auth](https://www.better-auth.com/)**: The most comprehensive authentication library for TypeScript.

## Forms & Validation
- **[React Hook Form](https://react-hook-form.com/)**: Performant, flexible and extensible forms with easy-to-use validation.
- **[Zod](https://zod.dev/)**: TypeScript-first schema declaration and validation library.
