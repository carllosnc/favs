# Contributing to Favs

First off, thanks for taking the time to contribute!

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone** your fork locally.
3.  **Create a branch** for your feature or bug fix.
4.  **Make your changes** and commit them.
5.  **Push** to your fork.
6.  **Submit a Pull Request** to the main repository.

## Reporting Bugs

If you find a bug, please create a new issue and include:
- A clear description of the bug.
- Steps to reproduce it.
- Expected behavior vs. actual behavior.

## Feature Requests

We welcome new ideas! Please open an issue to discuss your feature idea before implementing it.

## Development

Please follow the existing code style. We use **TypeScript** and **Tailwind CSS**.
Ensure all code is properly typed and components are reusable where possible.
